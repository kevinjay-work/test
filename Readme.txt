I used ReactJS, Material-UI as a front-end library and javascript.
For dynamic product information i used real time database(Firebase) that i created manually.
I recommend  bluehost for hosting.

Follow these steps to run the project:
1: go to your terminal and write command git clone <the Url that I've send>
2: write cd test
3: write npm install (nodejs required)
4: npm start
5: When the command fully completed go to your browser
6: write localhost:3000
7: your project in then running
